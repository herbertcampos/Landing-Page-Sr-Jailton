# Jailton Pinturas 🎨

Site institucional desenvolvido para divulgação de serviços de pintura residencial e comercial.

---

## 📋 Sobre o projeto

Este projeto foi criado para apresentar os serviços do profissional **Jailton Pinturas**, oferecendo uma presença online simples, moderna e profissional.

O site possui:

- Página responsiva
- Botão direto para WhatsApp
- Sessão de serviços
- Galeria de trabalhos
- Layout profissional
- Compatível com celular e computador

---

## 🚀 Tecnologias utilizadas

- HTML5
- TailwindCSS
- Responsividade Mobile

---

## 📱 Funcionalidades

- Contato rápido via WhatsApp
- Exibição de serviços
- Galeria de imagens
- Layout leve e rápido
- Fácil hospedagem gratuita
